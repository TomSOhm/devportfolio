Featured Projects

Capstone Project with Kering

Worked on optimizing supply chain replenishment strategies for a major luxury retailer. Explored both statistical and reinforcement learning methods to define rules for replenishing regional warehouses from the central warehouse, and stores from the regional warehouse, aiming to minimize average stock while controlling for missing sales. The project involved:

  • Statistical Modeling: Developed and evaluated rule-based and probabilistic models for inventory management.
  • Reinforcement Learning: Implemented RL algorithms to learn dynamic replenishment policies that adapt to sales patterns and stock levels.
  • Data Analysis & Simulation: Analyzed historical sales and stock data, and simulated various replenishment scenarios to assess performance.
This project provided hands-on experience in supply chain analytics, statistical modeling, and the application of reinforcement learning to real-world business problems.

Supply Chain Optimization Statistical Modeling Reinforcement Learning Simulation Data Analysis Python

Protein Classification Challenge

Led a comprehensive bioinformatics project to classify proteins into 18 functional categories using both their amino acid sequences and 3D structural graphs. I experimented with advanced machine learning and deep learning models, including:

  • Hierarchical Graph Pooling with Structural Learning (HGP-SL): Captured global graph structure for improved message passing and protein representation.
  • Fusion Model (GNN + MLP): Combined graph-based structural features and sequence-based embeddings for robust classification.
  • CNN & LSTM Fusion: Extracted local and long-range dependencies from protein sequences using deep learning.
  • CountVectorizer + Logistic Regression: Established strong baselines with n-gram encoding and interpretable models.
This project strengthened my expertise in graph neural networks, model fusion, feature engineering, and the integration of biological data for real-world applications.

Graph Neural Networks Hierarchical Graph Pooling Deep Learning CNN LSTM MLP Logistic Regression CountVectorizer TFIDF Feature Engineering Bioinformatics Python PyTorch

Bovine Embryo Survival Prediction

Developed a deep learning pipeline to predict the developmental state of bovine embryos at 192 hours after fertilisation (HAF) using time-lapse video data. The approach involved:

  • Frame Extraction: Selected key frames from embryo development videos at specific time points for analysis.
  • CNN + LSTM Architecture: Used a ResNet-based convolutional neural network to extract features from each frame, followed by an LSTM to model temporal dynamics across frames.
  • PyTorch Implementation: Built custom PyTorch datasets and training loops, leveraging transfer learning and sequence modeling for robust classification.
This project provided hands-on experience in video-based deep learning, temporal modeling, and practical application of neural networks for biomedical prediction tasks.

Deep Learning Video Analysis CNN LSTM PyTorch Transfer Learning Sequence Modeling Biomedical AI Python

Trading Stocks Market Prediction

Collect trading data using Yahoo finance API, Kafka, Hadoop, (Spark, Cassandra) and different online prediction methods (using river or other tools) to predict markets stocks of global companies. For several countries, we used major industries stock data (US Google and Apple, France BNP Paribas, China Alibaba, Switzerland Novartis). We compared the performances of online methods with batch regression models (LSTM and autoregressive model). We finally concluded on the performances on recent stock market data.

Machine Learning for Detection of Early-Stage Parkinson using Gait Analysis

Explored machine learning algorithms to detect early-stage Parkinson's disease using gait analysis data. The project involved:

  • Data Collection & Cleaning: Integrated and preprocessed multiple public datasets, ensuring high-quality input for modeling.
  • Modeling: Explored and compared several models, including Random Forest, XGBoost, Feed-Forward Neural Networks (FFNN), and Variational Autoencoders, to capture both linear and complex non-linear patterns in gait data.
  • Explainable AI (XAI): Applied XAI techniques to interpret Random Forest predictions, providing transparency and actionable insights for clinical use.
  • Evaluation & Stratification: Conducted thorough model evaluation, comparison, and patient stratification to assess diagnostic performance and generalizability.
This project deepened my expertise in medical data science, advanced machine learning, and explainable AI for healthcare applications.

Machine Learning Data Science Data Cleaning Random Forest XGBoost Feed-Forward Neural Network Variational Autoencoder Explainable AI (XAI) Model Evaluation Stratification Python
